# 그래프 모듈 설치
## 1. Folium
- 지도표시 모듈
- Anaconda prompt 오픈 - pip install folium 엔터 - 설치완료

## 2. Graphviz 설치
- url : https://gitlab.com/graphviz/graphviz/-/package_files/7097345/download
- 설치후 아나콘다 프롬프트에서 pip install graphviz 명령 실행
- 환경변수 설정
	- 사용자 path에는 bin까지
	- 시스템 path에는 dot.exe까지
